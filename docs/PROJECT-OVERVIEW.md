## 📖 项目简介和目标

### 项目简介

中国象棋项目是一款集**单机对战**、**联网对战**、**棋局学习**、**社交互动**于一体的现代化中国象棋平台。项目采用前后端分离架构，集成 Stockfish 最强开源象棋引擎，为用户提供从新手到大师的全段位 AI 对弈体验。

### 项目目标

| 目标 | 描述 |
|------|------|
| **全难度覆盖** | 集成 Stockfish 引擎，提供 10 个难度等级（500-2800 Elo） |
| **实时对战** | 基于 WebSocket 实现低延迟（P95 < 50ms）的联网匹配和好友对战 |
| **棋局生态** | 支持棋局保存、回放、分享，构建棋谱社区 |
| **成长体系** | 建立天梯排名、成就系统，激励用户持续提升 |
| **学习平台** | 整合新手教程、残局挑战、棋谱学习，降低入门门槛 |

### 目标用户

| 用户类型 | 特征 | 占比 |
|---------|------|------|
| **新手用户** | 刚接触象棋，需要了解规则、基础走法 | 30% |
| **休闲玩家** | 有一定基础，偶尔娱乐对战 | 40% |
| **竞技玩家** | 水平较高，追求天梯排名和胜率 | 20% |
| **学习用户** | 主动学习棋谱、残局，提升棋力 | 10% |

## 🚀 快速开始指南

### 1. 环境要求

- **Node.js**: v20+
- **Python**: 3.11+
- **Docker**: 24+ (可选，用于数据库)
- **Stockfish**: 16+ (AI 引擎)

### 2. 克隆项目

```bash
cd ~/.openclaw/workspace/projects
git clone <repository-url> chinese-chess
cd chinese-chess
```

### 3. 启动数据库（Docker）

```bash
docker-compose up -d postgres redis
```

### 4. 后端配置

```bash
cd src/backend

# 创建虚拟环境
python -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 配置数据库连接等

# 数据库迁移
python manage.py migrate

# 启动开发服务器
python manage.py runserver
```

### 5. 前端配置

```bash
# 用户端前端
cd src/frontend-user
npm install
npm run dev

# 管理端前端
cd src/frontend-admin
npm install
npm run dev
```

### 6. 访问应用

| 服务 | 地址 |
|------|------|
| **用户端** | http://localhost:3000 |
| **管理端** | http://localhost:3001 |
| **后端 API** | http://localhost:8000 |
| **API 文档** | http://localhost:8000/api/v1/docs/ |

---

### 7. 运行测试

```bash
# 后端测试
cd src/backend
pytest --cov=apps

# 前端测试
cd src/frontend-user
npm test
```
