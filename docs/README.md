## 📖 项目简介和核心价值

### 项目简介

中国象棋项目是一款集**单机对战**、**联网对战**、**棋局学习**、**社交互动**于一体的现代化中国象棋平台。项目采用前后端分离架构，集成 Stockfish 最强开源象棋引擎，为用户提供从新手到大师的全段位 AI 对弈体验。

### 核心价值

| 价值维度 | 描述 |
|---------|------|
| **全难度 AI 对战** | 集成 Stockfish 16 引擎，提供 10 个难度等级（500-2800 Elo） |
| **实时联网对战** | 基于 WebSocket 实现低延迟（P95 < 50ms）的联网匹配和好友对战 |
| **完整棋局生态** | 支持 FEN 格式、棋局保存、回放、分享，构建棋谱社区 |
| **天梯排名系统** | ELO 积分、段位评定、排行榜，激励用户持续提升 |
| **学习平台** | 新手教程、残局挑战、棋谱学习，降低入门门槛 |
| **社交互动** | 房间聊天、好友对战、观战功能，增强用户粘性 |

### 目标用户

| 用户类型 | 特征 | 占比 | 核心需求 |
|---------|------|------|---------|
| **新手用户** | 刚接触象棋，需要了解规则、基础走法 | 30% | 教程、低难度 AI、规则提示 |
| **休闲玩家** | 有一定基础，偶尔娱乐对战 | 40% | 快速匹配、中等难度 AI |
| **竞技玩家** | 水平较高，追求天梯排名和胜率 | 20% | 高难度 AI、精准匹配、复盘分析 |
| **学习用户** | 主动学习棋谱、残局，提升棋力 | 10% | 残局挑战、棋谱库、AI 分析 |

## 🚀 快速开始指南

### 环境要求

| 软件 | 最低版本 | 推荐版本 |
|------|---------|---------|
| Python | 3.9 | 3.11+ |
| Node.js | 18.x | 20.x |
| PostgreSQL | 13 | 15+ |
| Redis | 6 | 7+ |
| Stockfish | 15 | 16 |

### 后端安装

```bash
# 1. 克隆项目
cd projects/chinese-chess

# 2. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，配置数据库、Redis、密钥等

# 5. 数据库迁移
python src/backend/manage.py migrate

# 6. 启动开发服务器
python src/backend/manage.py runserver
```

### 前端安装

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，配置 API 地址

# 4. 启动开发服务器
npm run dev
```

### Docker 部署

```bash
# 1. 构建镜像
docker-compose build

# 2. 启动服务
docker-compose up -d

# 3. 查看日志
docker-compose logs -f

# 4. 停止服务
docker-compose down
```

### 验证安装

1. 访问 `http://localhost:5173` 打开前端
2. 访问 `http://localhost:8000/api/health/` 检查后端健康状态
3. 注册账号并登录
4. 创建 AI 对局测试游戏功能

### 开发流程

1. **Fork 项目** - 创建个人分支
2. **创建特性分支** - `git checkout -b feature/AmazingFeature`
3. **提交更改** - `git commit -m 'Add some AmazingFeature'`
4. **推送到分支** - `git push origin feature/AmazingFeature`
5. **开启 Pull Request** - 提交代码审查

- 遵循 [开发约束规范](./DEVELOPMENT-CONSTRAINTS.md)
