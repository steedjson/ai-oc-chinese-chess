## 📝 测试编写策略

### 1. 单元测试优先
- 测试纯函数和简单类
- 使用 mock 隔离依赖
- 覆盖边界条件

### 2. 集成测试补充
- 测试模块间交互
- 测试数据库操作
- 测试 API 端点

### 3. 异步测试处理
- 使用 pytest-asyncio
- 适当等待异步完成
- 避免过于复杂的异步链
