### API 测试详情

#### ✅ 通过的 API

1. **用户登录** (`POST /api/v1/auth/login/`)
   - 响应时间：188ms
   - 错误处理：正确返回 `INVALID_CREDENTIALS`
   - 状态码：401

2. **用户注册** (`POST /api/v1/auth/register/`)
   - 响应时间：42-246ms
   - 验证逻辑：正确检测重复用户名/邮箱
   - 必填字段：正确要求 `password_confirm`
   - 状态码：201 (创建成功)

3. **获取当前用户** (`GET /api/v1/auth/me/`)
   - 响应时间：8ms
   - 认证：正确验证 Bearer Token
   - 数据格式：完整返回用户信息
   - 状态码：200

4. **创建 AI 游戏** (`POST /api/v1/ai/games/`)
   - 响应时间：11ms
   - 参数验证：正确要求 `ai_level`, `ai_side`, `fen_start`
   - 状态码：201

5. **获取用户信息** (`GET /api/v1/users/{id}/`)
   - 响应时间：10ms
   - 数据完整：包含用户资料、统计数据
   - 状态码：200

#### ❌ 失败的 API

1. **创建 AI 对弈** (`POST /api/v1/games/ai/`)
   - **问题**: 返回 405 Method Not Allowed
   - **原因**: 前端调用端点错误，正确端点应为 `/api/v1/ai/games/`
   - **影响**: AI 对战功能无法使用
   - **修复建议**: 修改前端 API 调用路径

2. **获取排行榜** (`GET /api/v1/ranking/leaderboard/`)
   - **问题**: 返回 404 Not Found
   - **原因**: 该端点未实现或路由配置错误
   - **影响**: 排行榜页面无法加载数据
   - **修复建议**: 实现排行榜 API 端点或修正路由
